# Food Image Classification for Calorie Estimation

This project was developed as a machine learning and deep learning practice project focused on food image classification.

The project explores transfer learning techniques using MobileNetV2 to classify food categories from images.

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- scikit-learn

## Features

- Image preprocessing
- Data augmentation
- CNN-based classification
- Transfer learning experiments
- Model evaluation

## Results

The model achieved approximately 78% accuracy on the test dataset.

Through this project, I gained practical experience with:
- Deep learning workflows
- Transfer learning
- Image classification
- Model training and evaluation
